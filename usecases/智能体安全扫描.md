# 使用 ClawGuard 进行智能体安全扫描

## 概述
使用 OpenClaw + [ClawGuard](https://github.com/NeuZhou/clawguard) 在安装前自动扫描技能和插件，保护你的 AI 智能体免受提示注入、隐私泄露和供应链攻击。

## 功能特性
- **安装前安全扫描**：安装任何技能/插件前进行安全扫描
- **提示注入检测**：25+ 种检测模式，支持多种语言（英文/中文/日文/韩文）
- **隐私信息脱敏**：自动识别并处理电子邮件、社保号、API 密钥、信用卡号 - 完全本地运行
- **意图-行为不一致检测**：捕获 AI 智能体说一套做一套的情况
- **供应链安全**：检测域名欺骗、代码混淆、反向 shell
- **OWASP 覆盖**：涵盖所有 10 个 AI 智能体安全风险类别（ASI01-ASI10）

## 安装
```bash
npm install -g @neuzhou/clawguard
# 或者使用 npx（无需安装）
```

## 使用方法
```
"安装前扫描这个技能：npx @neuzhou/clawguard scan ./my-skill"
"检查此文本是否包含提示注入"
"在发送前对 AI 智能体输出进行隐私信息脱敏"
"对我的 OpenClaw 工作区进行全面安全审计"
```

## 真实示例
```bash
$ npx @neuzhou/clawguard scan ./suspicious-skill
```

## 为何重要
拥有 shell 访问权限、文件系统访问权限和 API 调用能力的 AI 智能体需要安全扫描。ClawGuard 是 AI 智能体的"杀毒软件" - 285+ 种威胁模式，229 项测试，开源且免费。
