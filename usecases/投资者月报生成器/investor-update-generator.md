# 投资者月报生成器

从 Stripe、Google Analytics 和你的 CRM 拉取关键指标，自动生成一份带有图表和叙述的精美月度投资者邮件。

## 痛点

写投资者月报很重要，但总是被排到优先级后面。你知道应该每月发送更新，但从 5 个不同工具中拉取数据并组织成一段叙述需要 3-4 小时。大多数创始人要么跳过某个月，要么发送低质量的内容。

## 功能介绍

- 从 Stripe 拉取收入数据（MRR、增长率、流失率）
- 从 Google Analytics 获取流量和转化数据
- 从你的 CRM 或 Notion 获取销售管道数据
- 将所有数据编译成一封结构化的投资者更新邮件
- 遵循经过验证的投资者更新模板（亮点、指标、挑战、求助）

## 所需技能

- [stripe](https://clawhub.ai) — 收入和账单数据
- [google-analytics](https://clawhub.ai) — 流量和转化指标
- Gmail — 用于发送更新邮件
- Google Sheets — 用于数据汇总（可选）

## 如何设置

1. 安装所需技能：
```bash
clawhub install stripe
clawhub install google-analytics
```

2. 向 OpenClaw 发送指令：
```text
每月 1 日上午 10 点，生成一封投资者更新邮件：

1. 从 Stripe 拉取：MRR、MRR 增长率、新增客户、流失客户、净收入留存率
2. 从 Google Analytics 拉取：独立访客数、注册转化率、Top 获客渠道
3. 从我的销售管道（Notion "Sales Pipeline" 数据库）拉取：管道中的交易、本月成交的交易

4. 按以下格式起草邮件：
   - 主题："[公司名称] — [月份] 投资者更新"
   - TL;DR（3 句话）
   - 关键亮点（前 3 项）
   - 指标仪表盘（表格格式）
   - 挑战与收获
   - 需要帮助的事项

5. 将草稿发送到 Telegram 供我审阅。
6. 我确认后，发送到我的投资者列表（存储在记忆中）。
```

## 实际案例

**你：** "起草本月的投资者更新"

**OpenClaw：**

**草稿已准备好供审阅：**

主题：Acme AI — 2026 年 2 月投资者更新

**TL;DR：** MRR 增长 18% 至 $47K。签下最大一笔企业订单（$2,400/月）。正在努力降低 SMB 细分的流失率。

**关键亮点：**
1. 🚀 MRR 达到 $47,200（月环比 +18%）
2. 🏢 签下首笔企业订单 — GlobalCorp，$2,400/月
3. 📈 SEO 优化后自然流量增长 34%

**核心指标：**
| 指标 | 本月 | 上月 | 变化 |
|--------|-----------|------------|--------|
| MRR | $47,200 | $40,000 | +18% |
| 客户数 | 312 | 287 | +8.7% |
| 流失率 | 3.2% | 2.8% | ⚠️ +0.4% |

**挑战：**
- SMB 流失率上升 — 正在调查新手引导体验

**需要帮助：**
- 介绍 [目标公司] 的产品副总裁

需要我将这封邮件发送到你的投资者列表吗？

## 变体

- **董事会演示生成器**：扩展为带有幻灯片的完整董事会演示文稿
- **每周内部报告**：面向团队的简短版本，每周一发送
- **基金 LP 报告**：为基金管理人向 LP 汇报修改模板

## 关键洞察

- 投资者普遍认为最好的创始人会定期发送诚实透明的更新 — 即使数据不理想
- "需要帮助"部分是最有价值的 — 它将一份报告变成了一个可执行的人脉请求
- 自动化数据收集意味着你的时间花在叙述和策略上，而不是处理电子表格
