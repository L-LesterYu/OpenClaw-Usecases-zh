# 第二大脑

你时常冒出灵感、发现有趣的链接、听说值得读的书——却始终没有一个好的方式来记录它们。Notion 变得越来越复杂，Apple Notes 成了上万条未读条目的坟场。你需要一个像给朋友发消息一样简单的方式。

这个工作流将 OpenClaw 变成一个记忆采集系统，你可以通过短信与它交互，背后还有一个可搜索的自定义界面，随时可以浏览。

## 功能概述

- 通过 Telegram、iMessage 或 Discord 随时给 OpenClaw 发送任何内容——"提醒我读一本关于本地大模型的书"——它会立刻记住
- OpenClaw 内置的记忆系统将你告诉它的所有内容永久存储
- 一个自定义的 Next.js 仪表板，让你可以搜索每一条记忆、对话和笔记
- 全局搜索（Cmd+K），涵盖所有记忆、文档和任务
- 没有文件夹、没有标签、没有复杂的组织方式——只有文本和搜索

## 痛点

每款笔记应用最终都会变成一种负担。你不再使用它，因为整理的阻力大于遗忘的阻力。核心洞察是：**记录应该像发消息一样简单，检索应该像搜索一样便捷**。

## 所需技能

- Telegram、iMessage 或 Discord 集成（用于文本采集）
- Next.js（OpenClaw 会为你构建——无需编程）

## 如何设置

1. 确保你的 OpenClaw 已连接到你偏好的消息平台（Telegram、Discord 等）。

2. 立即开始使用——只需给机器人发送你想记住的任何内容：
```text
嘿，提醒我读《设计数据密集型应用》
保存这个链接：https://example.com/interesting-article
记住：John 推荐了第五街的那家餐厅
```

3. 通过向 OpenClaw 发送提示来构建可搜索的界面：
```text
我想构建一个第二大脑系统，可以在其中查看我们所有的笔记、
对话和记忆。请用 Next.js 构建它。

包含：
- 所有记忆和对话的可搜索列表
- 全局搜索（Cmd+K）
- 按日期和类型筛选的功能
- 简洁、极简的界面
```

4. OpenClaw 会为你构建并部署整个 Next.js 应用。导航到它提供的 URL，你就拥有了你的第二大脑仪表板。

5. 从此以后，每当你想到什么——在路上、在会议中、睡前——只需给机器人发一条消息。需要查找什么的时候，回到仪表板即可。

## 核心洞察

- 强大之处在于**零阻力的记录**。你不需要打开一个应用、选择文件夹或添加标签。只需发消息。
- OpenClaw 的记忆系统是累积的——它记住你曾经告诉它的*所有内容*，随时间推移变得越来越强大。
- 你可以在手机上给机器人发消息，它会在你的电脑上构建内容。对话就是界面。
