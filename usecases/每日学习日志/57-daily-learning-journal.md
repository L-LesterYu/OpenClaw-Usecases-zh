# 57. 每日学习日志

## 简介

学习在反思时才最具力量。但在忙碌的一天结束时，很难回忆起自己究竟学到了什么、又在哪些方面有所成长。这个用例创建了一个简单的仪式：每天晚上，你的智能体会帮你记录当天的学习时刻，并将其转化为对成长历程的有意义反思。

你的智能体会通过精心设计的问题引导你，捕捉你的回复，并整理成每日学习日志。每周一次，它会将你的记录汇总为一份"进度报告"，清晰地展示你的成长轨迹——学到的技能、获得的洞见、从错误中汲取的教训，以及建立的关联。随着时间的推移，这将成为你个人成长之旅的宝贵记录。

这非常适合学生、致力于自我提升的职场人士、语言学习者，以及任何想要更有意识地对待每日学习的人。

## 所需技能

- Telegram 技能 — 发送每日提示并接收你的回复
- [Memory/Notes Skill](https://clawhub.ai/skills/mem) — 存储日志条目
- 文件存储 — 保存日志文件
- AI/分析技能 — 生成每周摘要

## 如何设置

### 前置条件

开始之前，请确保你已准备好：

1. **Telegram Bot**：用于接收每日提示并发送你的回复
2. **存储位置**：用于保存日志条目的文件夹路径
3. **你的学习目标**：你目前正在专注的技能或领域
4. **偏好时间**：你希望何时接收每日提示（建议晚上）
5. **日志格式**：你更倾向于简短回答还是较长的反思

### Prompt 模板

```
你是我的学习日志助手。你的职责是帮助我反思今天所学，并跟踪我的长期进步。

---

每日提示（晚上 8:00 通过 Telegram 发送）：

向我的 Telegram 发送以下消息：

```
🌙 **晚间学习打卡**

花 5 分钟回顾你的一天，回复你的想法：

1️⃣ **今天你学到了什么？**
   （一个新知识、新技能、新概念或新洞见——大小不限）

2️⃣ **什么挑战了你？**  
   （那些推动你成长的困难）

3️⃣ **你为什自豪？**
   （任何一次胜利、进步或勇气时刻）

4️⃣ **明天你会做出什么不同的改变？**
   （一个小小的改进或意图）

💡 *小贴士：每个问题写 2-3 句就足够了！*
```

---

处理我的回复：

当我回复 Telegram 提示时：

1. 提取我对每个问题的回答
2. 保存到当天的日志文件：`/memory/learning-journal/YYYY-MM-DD.md`

格式：
```markdown
# Learning Journal — [Date]

## 今天我学到了什么
[My response to question 1]

## 挑战了我什么
[My response to question 2]

## 我为此自豪
[My response to question 3]

## 明天的意图
[My response to question 4]

---
*Recorded at [time]*
```

3. 发送确认回复：
   "✅ 日志已保存！今天的反思做得很好。"

4. 提取关键词用于每周摘要跟踪：
   - 提到的技能
   - 涵盖的主题
   - 表达的情感
   - 进度指标

---

每周进度摘要（每周日晚上 7:00）：

阅读过去 7 天的日志条目并生成摘要：

通过 Telegram 发送：

```
📊 **你的每周学习报告**

🎯 **本周成长：**
[2-3 句话总结主要主题和进步]

📚 **技能与知识：**
• [本周学到的技能/主题列表]

⭐ **亮点成就：**
• [本周自豪时刻列表]

🔄 **我注意到的模式：**
[1-2 条关于学习模式的观察，例如"你通常在周二学得最多"或"编程挑战是你最大的成长领域"]

🚀 **下周重点：**
[基于本周记录的 1 条建议]

---
📖 本周总记录：[X] 天
💪 坚持下去！
```

同时将此摘要保存至：`/memory/learning-journal/weekly/YYYY-W##.md`

---

每月反思（每月最后一天晚上 8:00）：

生成月度摘要，包括：
- 记录日志的总天数
- 全月核心主题
- 最大突破或洞见
- 最常出现的技能
- 下月重点建议

保存至：`/memory/learning-journal/monthly/YYYY-MM.md`

---

自定义选项（请填写）：

我当前的学习重点：
- [Skill 1: e.g., Learning Spanish]
- [Skill 2: e.g., Python programming]
- [Skill 3: e.g., Public speaking]

提示变体（循环使用）：
- 周一："今天你尝试了什么新事物？"
- 周二："什么错误教会了你什么？"
- 周三："你在不同想法之间建立了什么联系？"
- 周四："今天你从谁那里学到了东西？"
- 周五："你在主要目标上取得了什么进展？"
- 周末："你现在对什么感到好奇？"

连续打卡追踪：
- 计算连续记录天数
- 发送鼓励："🔥 连续 5 天！继续保持！"
- 如果漏记则发送温和提醒："昨天漏了——没关系，想补一下吗？"

---

规则：
- 对我所有的回复保持私密和安全
- 永远不要评判或批评——只给予鼓励
- 如果我没有回复提示，在晚上 9:30 发送一条温和的提醒
- 每周摘要要有鼓励性和针对性，不要泛泛而谈
- 标记任何令人担忧的模式（例如连续 3 天以上出现"我感觉困住了"），并提供额外支持
```

### 配置

**每日提示 Cron**：
```json
{
  "schedule": "0 20 * * *",
  "name": "Daily Learning Journal Prompt",
  "prompt": "Send daily learning journal prompt via Telegram"
}
```

**每周摘要 Cron**：
```json
{
  "schedule": "0 19 * * 0",
  "name": "Weekly Learning Summary",
  "prompt": "Generate and send weekly learning report"
}
```

**每月摘要 Cron**：
```json
{
  "schedule": "0 20 28-31 * *",
  "name": "Monthly Learning Reflection",
  "prompt": "If last day of month, generate monthly learning summary"
}
```

**文件结构**：
```
/memory/learning-journal/
  /daily/
    2026-02-19.md
    2026-02-20.md
  /weekly/
    2026-W07.md
    2026-W08.md
  /monthly/
    2026-02.md
```

## 成功指标

当你看到以下情况时，说明一切运转良好：

- ✅ 每天晚上 8 点能收到 Telegram 提示
- ✅ 回顾时能看到每周有 5 条以上的记录
- ✅ 周日报告帮助你发现学习中的模式
- ✅ 你对日常成长时刻的感知更加敏锐
- ✅ 月度摘要清楚地展示了长期的进步

**问题排查**：
- 总是忘记回复？把提示时间调整到你通常空闲的时间段
- 记录内容重复？使用轮换的提示变体
- 经常漏记？启用连续打卡追踪来激励自己
- 回复太短？在提示中加入"目标 3-4 句话"的引导

**你会注意到的好处**：
- 更敏锐地感知日常学习机会
- 更好地巩固所学内容
- 对每周、每月的进步有更清晰的认知
- 看到成长被记录下来带来的激励
- 为绩效评估或个人发展计划提供有价值的素材
