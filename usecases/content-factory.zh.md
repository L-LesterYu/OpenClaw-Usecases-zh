# 多智能体内容工厂

你是一个内容创作者，每天都在多个平台上处理研究、写作和设计工作。每个步骤——寻找热门话题、撰写脚本、生成缩略图——都会耗费大量时间。如果有一支专业化智能体团队在夜间为你处理这一切呢？

这个工作流程在 Discord 内部建立了一个多智能体内容工厂，不同的智能体在不同的频道中处理研究、写作和视觉资产。

## 功能特点

- **研究智能体**每天早上扫描热门故事、竞品内容和社交媒体，寻找最佳内容机会
- **写作智能体**将精选的想法转化为完整的脚本、线程或新闻通讯草稿
- **缩略图智能体**为内容生成 AI 缩略图或封面图像
- 每个智能体在独立的 Discord 频道中工作，保持内容井然有序且可审查
- 按计划自动运行（例如每天早上 8 点），醒来即可获得完成的内容

## 解决痛点

内容创作包含三个阶段——研究、写作和设计，而大多数创作者都在手动处理所有三个环节。即使使用 AI 写作工具，你仍然需要逐一提示。这个系统将智能体串联成流水线，一个智能体的输出成为下一个智能体的输入，实现完全自动化的工作流程。

## 所需技能

- Discord 多频道集成
- `sessions_spawn` / `sessions_send` 多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似工具用于社交媒体研究
- 本地图像生成（例如 Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于基于 RAG 的研究）

## 如何设置

1. 设置一个 Discord 服务器（或者让 OpenClaw 为你设置——只需说"为我们设置一个 Discord"）。

2. 为每个智能体创建频道：
   - `#research` — 热门话题和内容机会
   - `#scripts` — 写作草稿和大纲
   - `#thumbnails` — 生成的图像和封面艺术

3. 提示 OpenClaw：
```text
我想要在 Discord 内部建立一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体 (#research)：每天早上 8 点，研究所在领域中表现最好的
   热门故事、竞品内容和社交媒体表现。发布前 5 个内容机会及其来源。

2. 写作智能体 (#scripts)：从研究智能体的最佳想法中选择一个，
   撰写完整的脚本/线程/新闻通讯草稿。发布到 #scripts。

3. 缩略图智能体 (#thumbnails)：为内容生成 AI 缩略图或封面图像。
   发布到 #thumbnails。

让他们的工作在不同的频道中有条理地组织。
每天早上自动运行这个流水线。
```

4. 根据你的平台进行定制：
```text
我专注于 X/Twitter 线程，而不是 YouTube。请将写作智能体
改为生成推文线程，而不是视频脚本。
```

## 关键洞察

- **链式智能体的力量**——研究驱动写作，写作驱动缩略图。你无需单独提示每个步骤。
- Discord 频道让你能够单独审查每个智能体的工作，并提供反馈，如"脚本太长"或"多关注 AI 新闻"。
- 你可以为任何内容格式调整此系统：推文、新闻通讯、LinkedIn 帖子、播客大纲、博客文章。
- 使用本地模型进行图像生成（如 Mac Studio 上的 Nano Banana）可以降低成本并让你获得更多控制权。
