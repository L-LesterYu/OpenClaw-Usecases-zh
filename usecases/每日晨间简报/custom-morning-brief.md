# 每日晨间简报

你每天醒来后的前30分钟都在忙着追赶进度——刷新闻、查看日历、翻看待办清单，试图弄清楚今天哪些事情最重要。如果所有这些都已经为你整理好，以一条短信的形式等着你呢？

这个工作流让 OpenClaw 每天在预定时间向你发送一份完全个性化的晨间简报，涵盖新闻、任务、创意和主动建议。

## 功能介绍

- 每天在同一时间（例如早上 8:00）通过 Telegram、Discord 或 iMessage 发送结构化的晨间报告
- 通过浏览网页，研究与你兴趣相关的隔夜新闻
- 查看你的待办清单，提取当天需要完成的任务
- 在你睡觉时生成完整的创意内容（完整脚本、邮件草稿、商业提案——而不仅仅是想法）
- 推荐当天 AI 可以自主帮你完成的任务

## 解决的痛点

你把最高效的清晨时光都花在了"了解情况"上。与此同时，你的 AI 代理整夜闲置无事可做。晨间简报将闲置的夜间时段转化为高效的前期准备——让你醒来时，工作已经完成。

## 所需技能

- Telegram、Discord 或 iMessage 集成
- Todoist / Apple 提醒事项 / Asana 集成（使用你常用的任务管理工具即可）
- [x-research-v2](https://clawhub.ai) 用于社交媒体趋势研究（可选）

## 如何设置

1. 将 OpenClaw 连接到你的消息平台和任务管理工具。

2. 向 OpenClaw 发送提示：
```text
我想设置一个定期的晨间简报。每天早上 8:00，
通过 Telegram 向我发送一份报告。

我希望这份报告包含以下内容：
1. 与我兴趣相关的新闻（AI、初创公司、科技）
2. 今天可以创作的创意内容建议
3. 我今天需要完成的任务（从我的待办清单中提取）
4. 你今天可以帮我完成的任务推荐

对于创意内容建议，请撰写完整的脚本/大纲草稿——而不仅仅是标题。
```

3. OpenClaw 会自动设置定时任务。第二天早上检查你的消息以确认是否正常工作。

4. 随时通过文字自定义调整——直接给你的机器人发消息：
```text
在我的晨间简报中添加天气预报。
停止包含一般新闻，只关注 AI。
每天早上加入一句励志名言。
```

5. 如果你不确定该加入哪些内容，不必担心——直接说：
```text
我希望这份报告包含与我相关的内容。
想一想什么内容放在这份报告中最有帮助。
```

## 关键要点

- AI 推荐任务部分是最强大的功能——它让代理主动思考如何帮助你，而不是被动等待指令。
- 你只需通过发消息就能自定义简报。说"在我的晨间简报中添加股价"即可更新。
- 完整草稿（而不仅仅是想法）是节省时间的关键。醒来看到的是脚本，而不只是建议。
- 无论你从事什么行业，一份包含任务、新闻和主动建议的晨间简报都是普遍有用的。
