# 域名注册助手

通过 OpenClaw 智能体在 1000+ 顶级域名（TLD）中注册域名。检查可用性、对比价格、完成注册——全部通过自然对话完成。

## 痛点

注册域名通常意味着要在臃肿的注册商网站之间反复跳转、在多个标签页中对比价格、还要手动填写表单。

## 功能介绍

让你的 OpenClaw 智能体处理整个域名注册流程：

- **检查可用性** — 在 1000+ 顶级域名中查询任意域名的注册状态
- **对比价格** — 跨不同顶级域名对比价格，找到最优方案
- **注册域名** — 直接通过对话完成域名注册
- **跟踪订单** — 查询注册状态和订单进展

## 所需技能

[lobsterdomains](https://clawhub.ai/esokullu/lobsterdomains) — 需要 LobsterDomains API 密钥（设置为 `LOBSTERDOMAINS_API_KEY`）。

在 [lobsterdomains.xyz](https://lobsterdomains.xyz) 获取 API 密钥。

## 设置方法

1. 从 [lobsterdomains.xyz](https://lobsterdomains.xyz) 获取 API 密钥
2. 将技能添加到你的智能体并配置 API 密钥
3. 开始聊天 — 让智能体帮你查找和注册域名

## 示例提示词
