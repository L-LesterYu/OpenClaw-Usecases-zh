# 收件箱清理

订阅邮件是占据收件箱空间的罪魁祸首。它们经常堆积如山，却从未被打开过。

## 所需技能
[Gmail OAuth 设置](https://clawhub.ai/kai-jar/gmail-oauth)。

## 如何设置
1. [可选] 为 OpenClaw 创建一个专属的 Gmail 邮箱。
2. [可选] 从你的主邮箱退订所有订阅邮件，然后使用 OpenClaw 邮箱重新订阅。
3. 安装技能并确保它能正常工作。
4. 向 OpenClaw 发送以下指令：
```txt
我希望你每天晚上8点运行一个定时任务，读取过去24小时内所有订阅邮件，给我一份最重要内容的摘要以及详细阅读的链接。然后询问我的反馈，了解你挑选的内容是否合适，并根据我的偏好更新你的记忆，以便在未来的任务中提供更好的摘要。
```
