# 62. 通过Telegram控制智能家居

## 简介

随时随地控制你的智能家居——只需发送一条Telegram消息。"关掉客厅灯。""将恒温器调到22°C。""前门锁了吗？"你的AI智能体充当智能家居系统的自然语言接口。

再也不用在多个设备App之间来回切换。一个聊天界面掌控一切。支持Home Assistant、SmartThings或任何提供API的系统。

## 所需技能

- Home Assistant — 智能家居集成
- Browser Control — 基于Web控制的备选方案

## 如何设置

### 前提条件
- 在你的网络中运行Home Assistant（或类似系统）
- Home Assistant API令牌
- 连接到OpenClaw的Telegram机器人

### 提示词模板
```
你是我的智能家居控制器。我会通过Telegram用自然语言发送命令。

已连接系统：
- Home Assistant 地址：http://[YOUR_HA_IP]:8123
- API令牌：[YOUR_TOKEN]

需要理解的命令示例：
- "关掉所有灯"
- "把卧室调到21度"
- "车库门开了吗？"
- "开启影院模式"（调暗灯光，打开电视）
- "晚安"（锁门、关灯、恒温器调至18°C）
- "我出门了"（启动安防、关灯、恒温器调至节能模式）

我使用的场景：
- "影院模式"：客厅灯光20%，电视开启
- "早安"：厨房灯开，咖啡机启动，恒温器22°C
- "晚安"：全部灯光关闭，门锁上，恒温器18°C

规则：
- 每次操作后用简短回复确认
- 如果命令有歧义，请求澄清
- 未经二次确认绝不解锁门
- 当"离家模式"激活时，如有门窗传感器触发则立即发送告警
```

### 配置
- 常驻运行：即时响应Telegram消息
- 安全告警：当"离家模式"激活时，每5分钟进行一次心跳检查

## 成功指标
- 通过一个聊天界面控制整个家庭
- 响应时间低于5秒
- 安全告警稳定可靠
