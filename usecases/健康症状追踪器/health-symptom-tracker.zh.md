# 健康症状追踪器

识别食物敏感症需要持续记录一段时间，这很容易让人厌烦。你需要提醒来记录，以及分析来发现模式。

这个工作流程自动跟踪食物和症状：

• 在专门的 Telegram 主题中发送你的食物和症状信息，OpenClaw 会自动记录所有内容并带有时间戳
• 每日3次提醒（早上、中午、晚上）提示你记录餐食
• 随着时间推移，分析模式以识别潜在的触发因素

## 你需要的技能

- 用于提醒的定时任务
- 用于记录的 Telegram 主题  
- 文件存储（Markdown 日志文件）

## 如何设置

1. 创建一个名为 "health-tracker"（或类似名称）的 Telegram 主题。
2. 创建日志文件：`~/clawd/memory/health-log.md`
3. 提示 OpenClaw：
```text
当我在 "health-tracker" 主题中发送消息时：
1. 解析消息中的食物项目和症状
2. 记录到 ~/clawd/memory/health-log.md 并带有时间戳
3. 确认记录了什么

设置3个每日提醒：
- 上午8点："🍳 记录你的早餐"
- 下午1点："🥗 记录你的午餐"
- 晚上7点："🍽️ 记录你的晚餐和任何症状"

每个星期天，分析过去一周的日志并识别模式：
- 哪些食物与症状相关？
- 是否有特定时间模式？
- 是否有明确的触发因素？

将分析结果发布到 health-tracker 主题。
```

4. 可选：为 OpenClaw 添加一个内存文件来记录已知的触发因素，并在模式出现时更新它。
