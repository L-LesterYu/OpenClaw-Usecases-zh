# 通过聊天进行AI视频编辑

编辑视频通常意味着打开时间线编辑器，拖拽片段，点击菜单。对于重复性编辑——修剪片头、为一批片段添加字幕、调整10个视频的颜色——这种手动循环会浪费大量时间。

这个用例将视频编辑变成了一场对话。描述你想要进行的修改，发送文件，然后获得结果回来。无需时间线，无需 GUI。

## 你可以做什么

- 通过简单语言描述时间戳来修剪、剪切和合并片段
- 添加带自动音频降噪的背景音乐
- 从语音生成并烧录字幕（支持50多种语言）
- 调色片段（"让它更温暖"，"匹配第一个片段的色调"）
- 裁剪为适合TikTok/Reels/Shorts的竖屏格式
- 用相同的编辑批量处理多个文件

## 你需要的技能

- [video-editor-ai](https://clawhub.ai/skills/video-editor-ai) — 基于聊天的视频编辑，支持背景音乐、字幕和导出
- [ai-subtitle-generator](https://clawhub.ai/skills/ai-subtitle-generator) — 自动字幕生成、字幕烧录、SRT导出

## 如何设置

1. 安装技能：
```bash
clawhub install video-editor-ai
clawhub install ai-subtitle-generator
```

2. 将视频文件发送到聊天中并描述你的编辑要求：
```text
将这个视频从 0:15 修剪到 1:30，添加背景音乐（欢快的音乐），
并烧录英文字幕。
```

3. 对于批量处理，描述模式：
```text
我在 /videos/raw/ 目录下有5个片段。对于每个片段：
- 裁剪为 9:16 竖屏格式
- 在底部添加自动生成的字幕
- 导出为 mp4
```

智能体会处理 API 调用，轮询完成状态，并将处理完成的文件发送回聊天。

## 使用技巧

- 明确时间戳和输出格式（"导出为1080p的mp4"）
- 对于字幕工作，如果不是英语，请注明源语言
- 调色在使用描述性参考（"温暖的日落色调"）时比使用技术参数效果更好
- 避免上传敏感素材（人脸、身份证、机密屏幕），除非你已经审查过提供商的数据保留和隐私政策
