# 通过对话进行 AI 视频剪辑

剪辑视频通常意味着打开时间线编辑器，拖拽片段，在菜单中点击操作。对于重复性编辑——裁剪片头、为一批片段添加字幕、调整 10 个视频的色调——这种手动流程会耗费大量时间。

这个用例将视频剪辑变成了一场对话。描述你想要的修改，拖入文件，就能拿到结果。没有时间线，没有图形界面。

## 你可以做什么

- 用自然语言描述时间戳来裁剪、剪切和合并片段
- 添加背景音乐并自动降低原声音量（音频闪避）
- 从语音生成并烧录字幕（支持 50+ 种语言）
- 调色（"让画面更暖"、"匹配第一个片段的风格"）
- 裁剪为竖屏，适配 TikTok/Reels/Shorts
- 批量处理多个文件，应用相同编辑

## 所需技能

- [video-editor-ai](https://clawhub.ai/skills/video-editor-ai) — 基于对话的视频剪辑，支持背景音乐、字幕和导出
- [ai-subtitle-generator](https://clawhub.ai/skills/ai-subtitle-generator) — 自动字幕生成、字幕烧录、SRT 导出

## 如何设置

1. 安装技能：
```bash
clawhub install video-editor-ai
clawhub install ai-subtitle-generator
```

2. 将视频文件拖入聊天并描述你的编辑需求：
```text
将这个视频从 0:15 裁剪到 1:30，添加背景音乐（轻快的风格），
并烧录中文字幕。
```

3. 对于批量处理，描述编辑模式：
```text
我在 /videos/raw/ 目录下有 5 个片段。对每一个执行：
- 裁剪为 9:16 竖屏
- 在底部添加自动生成的字幕
- 导出为 mp4
```

智能体会处理 API 调用、轮询完成状态，并将完成的文件返回到聊天中。

## 提示

- 明确指定时间戳和输出格式（"以 1080p 导出为 mp4"）
- 字幕工作方面，如果源语言不是英语，请说明源语言
- 调色效果使用参考描述（"温暖的日落色调"）效果最佳，而非技术参数值
- 避免上传敏感视频素材（人脸、身份证、机密屏幕），除非你已审查了服务提供商的数据保留和隐私政策
