# X 账号分析

目前有很多网站可以为你提供 X 账号的定性分析。虽然 X 自带了**数据分析（Analytics）**功能，但它更侧重于展示你的表现数据。

而定性分析关注的是你发布内容的质量，而非表现指标。通过这类分析，你可以获得以下洞察：
- 我的热门帖子有什么共同特征？
- 哪些话题能带来最多的互动？
- 为什么有时帖子能获得 1000+ 个赞，有时却不到 5 个赞？我做错了什么？

市面上有许多提供 X 数据分析的网站和应用，但它们大多聚焦于统计数据。可能只有 1-2 个网站支持与 AI 对话来帮你理解自己的账号表现。

而现在，你可以直接使用 OpenClaw 来完成这项分析，无需为这些网站支付每月 10-50 美元的订阅费。

## 所需技能

Bird 技能。`clawhub install bird`（已预装）

## 配置方法

操作流程如下：
1. 确认 Bird 技能正常工作。
2. 出于安全考虑，建议为你的 ClawdBot 创建一个独立的 X 账号。
3. 授权你的 X 账号：在 Chrome/Brave 浏览器中登录 x.com，并提供正确的 Cookie 信息（`auth-token`、`ct0`），使 OpenClaw 能够访问你的账号。
4. 让 OpenClaw 查看你的真实账号，抓取最近 N 条推文，然后向它提出任何你想了解的问题。你也可以让它编写特定的分析脚本。
