# 预约与订位助手

## 简介

找个水管工很容易，但真正预约到才是难事——打电话、排队等待、协调时间、确认信息。你的 AI 智能体可以帮你处理整个预约流程。

只需告诉你的智能体"我下周二需要看牙医"或"帮我订周五晚上一家意大利餐厅两人位"。它会搜索、比较选项并完成预订——通过网页表单、订位平台，甚至电话（通过文字预订系统）。

目标：你只需发出一个请求，预约就会自动出现在你的日历上。

## 所需技能

- [Web Search](https://clawhub.ai/skills/searching-assistant) — 搜索服务提供商
- Browser Control（浏览器控制） — 导航订位网站
- Web Fetch（网页抓取） — 读取商家详情

## 如何设置

### 前置条件
- 已连接 Telegram 机器人到 OpenClaw
- 已启用浏览器功能
- 你的大致位置和偏好设置

### 提示模板
```
你是我的预约助手。当我要求你预约某项服务时：

1. **搜索**：在 [你的城市/区域] 附近查找可用选项
2. **比较**：向我展示前 3 个选项，包括：
   - 名称、评分、价格范围
   - 符合我请求的可用时间段
   - 距离我的位置的距离
3. **预订**：我选定后，完成预约：
   - 通过浏览器填写在线订位表单
   - 如果没有在线订位，提供电话号码和通话话术供我使用
4. **确认**：将预约添加到我的日历，包含：
   - 地点和地址
   - 确认编号
   - 用于更改/取消的联系方式

我会要求预约的类型：
- 餐厅订位
- 医生/牙医预约
- 美发沙龙预约
- 上门服务（水管工、电工、保洁）
- 汽车维修/保养

我的偏好：
- 位置：[你的区域]
- 预算：[适中/灵活/经济实惠]
- 首选时间：[工作日晚间、周末等]

在最终确认任何预订之前，请先征得我的同意。
```

### 配置
- 按需触发：在你提出请求时响应
- 无需自动定时调度

## 成功指标
- 从发出请求到完成预订不超过 10 分钟
- 无需反复沟通——智能体处理所有后勤事务
- 预约自动出现在你的日历上
