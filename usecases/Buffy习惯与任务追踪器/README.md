# Buffy Agent — 习惯、任务与日常追踪器

你有要养成的习惯、要完成的任务、要坚持的日常——但它们散落在不同的应用中，彼此之间互不连通，也无法在你已有的工作流中随叫随到。Buffy Agent 通过直接运行在 OpenClaw（及你的其他频道）中作为个人行为引擎来解决这一问题，将日常对话转化为结构化的习惯追踪、任务管理和日常提醒。

## 痛点

习惯和任务应用都需要你主动打开它们。提醒很容易被忽略。真正的痛点在于上下文切换——你使用的每个工具都存在于不同的地方。Buffy Agent 同时运行在 OpenClaw、Telegram、Slack 和 ChatGPT 中，让你的习惯和任务跟随你到任何工作场景。

## 功能介绍

- **自然语言交互** — 用日常语言创建习惯、任务和日常安排：「每2小时提醒我喝水」或「添加一个任务：下午5点前完成PR评审」
- **习惯追踪** — 建立周期性行为并跟踪完成情况
- **任务管理** — 通过对话创建、更新、完成和优先级排序任务
- **日常与提醒调度** — 基于时间的通知直接发送到你偏好的频道
- **每日简报** — 定时汇总待办任务、活跃习惯和即将到来的日常安排
- **多频道同步** — 在 OpenClaw、Telegram、Slack 和 ChatGPT 之间保持单一数据源——一处更新，处处可见

## 所需技能

- [Buffy Agent OpenClaw 技能](https://clawhub.ai/phantue2002/buffy-agent) — 单一技能，一个 API 密钥（`BUFFY_API_KEY`）
- 调度 / cron 用于每日简报和定时提醒
- 可选：Telegram 或 Slack 集成用于跨频道同步

## 如何设置

1. 从 [ClawHub](https://clawhub.ai/phantue2002/buffy-agent) 安装 Buffy Agent 技能并添加你的 `BUFFY_API_KEY`。

2. 用日常语言开始追踪习惯：
```text
创建一个习惯：每天喝8杯水。
每天早上8点提醒我做10分钟冥想。
追踪我的习惯：睡前阅读30分钟。
```

3. 通过对话管理任务：
```text
添加一个任务：周五前完成API集成。
把"评审PR"标记为已完成。
显示我今天所有的待办任务。
```

4. 设置日常安排和每日简报：
```text
每天早上7:30，给我发送一份简报，包含：
- 今天的待办任务
- 活跃习惯及其状态
- 今天安排的日常事项
```

5. 可选 — 通过 OpenClaw 的集成连接 Telegram 或 Slack 来启用跨频道同步，然后告诉 Buffy：
```text
把我的任务和习惯也同步到Telegram。
如果我在Slack中标记某项为已完成，请在所有地方同步更新。
```

## 关键洞察

- **单一 API 端点**（`POST /v1/message`）使技能保持轻量——无需复杂配置，只需一个凭证即可运行。
- **行为记忆**意味着 Buffy 会随时间适应：它会学习你倾向于跳过打卡的时间段，并相应调整提醒时机。
- **多频道同步**是真正的差异化优势——你不会被锁定在单一界面中。在 OpenClaw 中更新任务列表，在 Telegram 中接收提醒，在 Slack 中查看回顾。
- 保持习惯聚焦：3–5个活跃习惯比试图同时追踪所有事情效果更好。
