# 多智能体内容工厂

你是一名内容创作者，需要在多个平台之间协调研究、写作和设计工作。每个步骤——寻找热门话题、编写脚本、生成缩略图——都会耗费你数小时的时间。如果有一个专业智能体团队在一夜之间完成所有工作呢？

这个工作流程在 Discord 内部设置了一个多智能体内容工厂，不同的智能体在专门的频道中处理研究、写作和视觉资产。

## 功能概述

- **研究智能体** 每天扫描热门故事、竞品内容和社交媒体，寻找最佳的内容机会
- **写作智能体** 接收最热门的想法，编写完整的脚本、推文线程或通讯草稿
- **缩略图智能体** 为内容生成 AI 缩略图或封面图片
- 每个智能体在自己的 Discord 频道中工作，保持所有内容井然有序且可审查
- 按计划自动运行（例如每天早上 8 点），让你醒来时就能看到完成的内容

## 痛点

内容创作分为三个阶段——研究、写作和设计——大多数创作者都是手动完成这三个阶段。即使使用 AI 写作工具，你仍然需要一个一个地提示它们。这个系统将智能体串联成一个流水线，前一个智能体的输出传递给下一个智能体，完全无需人工干预。

## 所需技能

- 支持多频道的 Discord 集成
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似的社交媒体研究工具
- 本地图像生成工具（例如 Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于基于 RAG 的研究）

## 如何设置

1. 设置一个 Discord 服务器（或者让 OpenClaw 为你设置，只需说"为我们设置一个 Discord"）。

2. 为每个智能体创建频道：
   - `#research` —— 热门话题和内容机会
   - `#scripts` —— 编写的草稿和大纲
   - `#thumbnails` —— 生成的图像和封面艺术

3. 向 OpenClaw 提示：
```text
我想你在 Discord 内部为我构建一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体 (#research)：每天早上 8 点，研究热门故事、竞品内容和
   我的领域中在社交媒体上表现良好的内容。发布前 5 个内容机会及其来源。

2. 写作智能体 (#scripts)：从研究智能体的最佳想法中
   编写完整的脚本/线程/通讯草稿。发布到 #scripts 中。

3. 缩略图智能体 (#thumbnails)：为内容生成 AI 缩略图或封面图片。
   发布到 #thumbnails 中。

让他们的工作都在不同的频道中有条理地组织。
每天早上自动运行这个流水线。
```

4. 根据你的平台进行定制：
```text
我专注于 X/Twitter 线程，而不是 YouTube。改变写作智能体
以生成推文线程而不是视频脚本。
```

## 核心洞察

- 强大之处在于 **串联的智能体** —— 研究驱动写作，写作驱动缩略图。你不必单独提示每个步骤。
- Discord 频道让你可以分别审查每个智能体的工作，并提供反馈，比如"脚本太长"或"多关注 AI 新闻"。
- 你可以将其适应任何内容格式：推文、通讯、LinkedIn 帖子、播客大纲、博客文章。
- 在本地运行图像生成模型（例如 Mac 上的 Nano Banana）可以降低成本并让你获得更多控制权。
