# 竞品监控仪表盘

监控竞品网站、定价页、更新日志和社交账号的变化——获取每周差异报告。

## 痛点

你总是从客户口中才得知竞品的动态，而不是第一时间发现。每周手动检查 5-10 个竞品网站、更新日志和社交账号既繁琐又不可靠。等你注意到变化时，已经是旧闻了。

## 功能介绍

- 每周爬取竞品网站和定价页面，自动检测任何变化
- 监控竞品更新日志和发布说明中的新功能
- 追踪竞品社交媒体上的产品公告和定位变化
- 生成差异报告，突出显示变化内容、新增功能及其对你的影响
- 归档历史快照，用于趋势分析

## 所需技能

- 浏览器自动化 — 用于爬取竞品页面
- 网页搜索 — 用于查找竞品公告
- [x-engagement](https://clawhub.ai) — 追踪竞品社交账号
- Google Sheets 或 Notion — 用于存储历史数据

## 设置方法

1. 向 OpenClaw 发送以下提示：
```text
我希望你每周监控以下竞品：
- [竞品 1]：website.com，twitter.com/handle，更新日志 URL
- [竞品 2]：website.com，twitter.com/handle，更新日志 URL
- [竞品 3]：website.com，twitter.com/handle，更新日志 URL

每周日晚上 8 点：
1. 访问每个竞品的定价页面，与上周快照进行对比
2. 检查他们的更新日志/博客中的新发布内容
3. 扫描他们的 X 账号中的产品公告
4. 生成一份"竞品情报报告"，包含：
   - 定价变化（旧价 vs 新价）
   - 新发布的功能
   - 定位或传播话术的变化
   - 值得关注的社交媒体动态
5. 通过 Telegram 将报告发送给我
6. 将快照保存到记忆中用于历史追踪
```

## 实际案例

**你：** "检查一下本周竞品的情况"

**OpenClaw：**

**每周竞品情报报告 — 2026年2月24日**

**竞品 A（Acme SaaS）：**
- 🔴 定价变化：专业版计划从 $29 涨至 $39/月
- 🟡 新功能：新增 AI 驱动的报表仪表盘
- 社交媒体：在 X 上宣布完成 B 轮融资（1.2 万赞）

**竞品 B（RivalCo）：**
- 🟢 定价无变化
- 🟡 新功能：Slack 集成上线
- 社交媒体：正在大量招聘——发布 8 个新工程岗位

**竞品 C（BigPlayer）：**
- 🔴 推出免费版（此前仅付费）
- 🟡 博客文章："为什么我们要走向开源"——潜在的战略转变
- 社交媒体：CEO 做了一期关于下沉市场的播客

**关键洞察：**
- 竞品 A 涨价 = 以性价比进行定位的机会
- 竞品 C 推出免费版 = 他们可能正在瞄准你的客户群体

## 变体

- **实时告警**：对于关键竞品，每天监控而非每周
- **招聘追踪器**：监控竞品招聘信息以推断战略方向
- **专利/商标监控**：追踪竞品的新知识产权申请

## 核心洞察

- 定价页面变化是最具价值的竞争情报——它们揭示了战略意图
- 招聘信息能提前 6 个月告诉你竞品正在构建什么
- 历史追踪能揭示规律——例如，竞品总是在融资公告前发布新功能
