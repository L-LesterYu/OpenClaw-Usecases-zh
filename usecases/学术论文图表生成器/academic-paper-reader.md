# 学术论文图表生成器

学术论文中的表格和图表往往包含宝贵的数据，但提取和重新可视化这些信息非常繁琐。手动从密集的 PDF 或 LaTeX 源码中重新绘制图表既耗时又容易出错。

本用例将 OpenClaw 变成一个图表提取与生成助手，能够阅读学术论文，从图表/表格中提取底层数据，并生成全新的可定制图表，用于进一步分析或展示。

## 功能特点

- 阅读完整论文（PDF 或 arXiv 源码），自动检测所有图表和表格
- 使用视觉和 OCR 模型从图表、图形和表格中提取数据点
- 支持使用你偏好的风格（如 Matplotlib、Plotly）重新生成图表
- 支持将提取的数据导出为 CSV 或 JSON 格式，便于进一步分析
- 提供原始图表与重新生成图表的并排对比

## 痛点

你需要快速复用或重新解读已发表论文中的数据，但手动提取速度慢且容易出错。自动化图表生成能够节省时间并确保准确性。

## 所需技能

- [arxiv-reader](https://github.com/Prismer-AI/Prismer/tree/main/skills/arxiv-reader)（或同等的论文获取技能）
- OpenClaw 中的视觉模型集成（用于图表解读）
- 可选：PDF 解析/OCR 集成（用于非 arXiv 论文）
- 绘图库集成（如 Matplotlib、Plotly）

## 设置方法

1. 将 OpenClaw 连接到你的论文来源（arXiv 或上传的 PDF），并确保启用了视觉模型。
2. 向 OpenClaw 发送提示：
```text
你是我的学术论文图表提取助手。
对于我分享的每篇论文：
1) 检测所有包含数据的图表和表格。
2) 从每个图表或表格中提取底层数据点。
3) 使用 Matplotlib 或 Plotly 重新生成图表。
4) 将提取的数据导出为 CSV 和 JSON 格式。
5) 提供原始图表与重新生成图表的并排对比。
```
3. 用一个具体的请求试一试：
```example
提取这篇论文中的所有图表，并用 Plotly 重新生成。
将图3和表2的数据导出为 CSV。
```

## 使用技巧

- 可以要求对每个图表/表格的数据提取准确性提供置信度评分。
- 请求提供可编辑的图表代码，以便进一步自定义。
- 对于复杂图表，可以让 OpenClaw 在提取的数据中标注任何不确定之处或近似值。
