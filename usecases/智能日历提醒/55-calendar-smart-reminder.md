# 55. 智能日历提醒

## 简介

标准的日历提醒太蠢了——只是在活动开始前15分钟响一声。你的AI智能体能做得好得多：它会检查你的日历，提前2小时发送带上下文的提醒，建议准备工作，甚至提醒你交通时间。

这个用例将你的智能体变成一个主动的日历助手，它能理解你的日程安排，预判你的需求，并通过Telegram发送带有可操作上下文的智能提醒。

再也不打无准备的仗了。

## 所需技能

- [日历访问](https://clawhub.ai/skills/calendar) — Google日历或Outlook集成
- 天气 — 用于户外活动准备

## 如何设置

### 前置条件
- Google日历或Outlook已连接到OpenClaw
- 已配置Telegram机器人用于通知

### 提示词模板
```
你是我的智能日历助手。在清醒时段（早上8点至晚上10点）每30分钟检查一次我的日历。

对于未来2小时内的每个即将到来的活动：

1. **上下文**：这个会议是关于什么的？有哪些人参加？
2. **准备工作**：我应该准备或携带什么？
   - 如果是客户会议 → 提醒我上次的互动情况
   - 如果是医生预约 → 提醒我带保险卡
   - 如果是聚餐 → 查看餐厅信息并建议穿搭
3. **后勤安排**：
   - 到那里需要多长时间？（如果是线下会议，查看路况）
   - 在下雨吗？需要带伞吗？
4. **冲突检测**：如果有两个活动时间重叠，标记提醒

通过Telegram发送提醒。格式：
📅 [活动名称] 2小时后开始
📍 [地点/链接]
📋 [准备事项]
⏱️ [建议在X:XX出发以准时到达]
```

### 配置
- 心跳检查：早上8点至晚上10点，每30分钟一次
- 发送提醒：每个活动开始前2小时

## 成功指标
- 零遗漏的预约
- 每次会议前都有上下文准备
- 交通时间预警防止迟到
