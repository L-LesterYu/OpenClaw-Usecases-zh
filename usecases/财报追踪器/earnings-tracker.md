# AI 驱动的财报追踪器

跟踪数十家科技公司的财报季意味着需要查看多个信息源并记住报告日期。你希望在无需手动逐一追踪每家公司的情况下，及时掌握 AI/科技领域的财报动态。

此工作流自动化了财报追踪与推送流程：

• 每周日预告：扫描即将到来的财报日历，将相关的科技/AI 公司发布到 Telegram
• 你选择关注的公司后，OpenClaw 会为每个财报日期安排一次性定时任务
• 每份财报发布后，OpenClaw 搜索财报结果，整理详细摘要（超预期/不及预期、关键指标、AI 亮点），并推送给您

## 所需技能

- `web_search`（内置）
- OpenClaw 定时任务支持
- 用于财报更新的 Telegram 话题

## 如何设置

1. 在 Telegram 中创建一个名为"earnings"（财报）的话题，用于接收更新。
2. 向 OpenClaw 发送以下提示词：
```text
每周日晚上6点，执行定时任务：
1. 搜索下周科技和AI公司的财报日历
2. 筛选出我关注的公司（NVDA、MSFT、GOOGL、META、AMZN、TSLA、AMD 等）
3. 将列表发布到我的 Telegram "earnings" 话题
4. 等待我确认要追踪哪些公司

当我回复要追踪的公司时：
1. 为每个财报日期/时间安排一次性定时任务
2. 每份财报发布后，搜索财报结果
3. 整理摘要，包括：超预期/不及预期、营收、每股收益、关键指标、AI相关亮点、业绩指引
4. 发布到 Telegram "earnings" 话题

记住我通常追踪的公司，以便每周自动推荐。
```
