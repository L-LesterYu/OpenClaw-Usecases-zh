# OpenClaw Usecases 中文版

## 项目简介

本项目旨在将 OpenClaw 的各种 usecase 案例翻译成中文，为中文用户提供本地化的使用指南和示例。

## 已支持的技能列表

| 场景名称 | 简短中文描述 | 文件路径 |
|---------|-------------|---------|
| 财报追踪器 | AI驱动的财报追踪与摘要推送，实时掌握科技和AI公司财报动态 | usecases/财报追踪器 |
| 预构建想法验证器 | 在开始构建新项目前自动验证想法竞争激烈度，避免重复开发，提高成功率 | usecases/预构建想法验证器 |
| 本地CRM框架 | 将OpenClaw转换为完全本地化的CRM系统，无需数据库搭建，一键安装 | usecases/本地CRM框架 |
| 目标驱动自主任务 | 让AI智能体主动理解你的目标，每天生成并执行推进目标的任务 | usecases/目标驱动自主任务 |
| 自动化会议纪要与行动项 | 自动监听会议记录，提取关键决策和行动项，创建Jira/Linear/Todoist任务 | usecases/自动化会议纪要与行动项 |
| 语义记忆搜索 | 为OpenClaw的markdown记忆文件添加基于向量的语义搜索功能，支持按含义查找记忆 | usecases/语义记忆搜索 |
| 每日Reddit摘要 | 每天运行摘要，为你提供来自喜爱子版块的热门帖子，支持浏览、搜索、提取评论线程等功能 | usecases/每日Reddit摘要 |
| 自主教育游戏开发流水线 | 自主管理教育游戏的创建和维护生命周期，采用"Bug优先"政策，每7分钟可生产1个新游戏或修复，支持0-15岁儿童的教育游戏开发 | usecases/自主教育游戏开发流水线 |

## 贡献指南

欢迎贡献翻译和新的 usecase 案例！

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'feat: add translated usecase 场景名称'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request
